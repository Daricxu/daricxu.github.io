Photocopier Interface Re-Design
Year of 2018
Student Group Project
Goal of this project is to improve the user experience of an existing photocopier in the school office via user research and new interface design. The challenge is that the existing photocopier has both touch screen and physical keyboard.
Project Deliverables
My Role: Task observation, information architecture design, UI refinement, final demo design

Research and Design Process
Heuristic Research - Task Observation
Task: Scan a file and send it to an email address
Three subjects are asked to perform the designed task in ten minutes. We documented the process of how they accomplished the task and the final results via video record. While the subject is working on the task, they are requested to think loudly.

Heuristic Research - Summary of Findings
Design Ideation - Information Architecture
The main task was separated into two operations: Scan File and Choose Destination for the scanned file. Then we came up with three type of information architecture by integrating them in different logical way.With the first two concepts, users need to follow step by step instructions to finish the whole task, either scan the file first or setup destination first. The third concept provides users all the options at one time and let the users choose where to start.
